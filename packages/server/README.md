# Noteum Server

## Overview
This is the server package for Noteum, a monorepo project using Bun runtime and Elysia framework.

## Prerequisites
- Node.js >= 22.0.0
- Bun >= 1.0.0

## Getting Started

### Installation
```bash
bun install
```

### Development
To start the development server:
```bash
bun run dev
```

### Production Build
```bash
bun run build
```

### Running Tests
```bash
bun test
```

## Configuration
- Runtime: Bun
- Framework: Elysia
- Port: 3000

## Features
- High-performance server using Bun runtime
- Elysia framework for efficient routing
- TypeScript support

Configure your environment variables in `.env` file.
