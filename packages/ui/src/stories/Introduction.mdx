# @noteum/ui - React Flow 组件库

欢迎使用 @noteum/ui React Flow 组件库！这是一个专为构建图形化工作流界面而设计的组件集合。

## 🎯 设计理念

本组件库基于 [React Flow](https://reactflow.dev) 构建，专注于为 AI 应用和工作流系统提供易用、美观的图形界面组件。

### 核心特性

- **🎨 开箱即用** - 预设计的节点和连接线样式
- **🤖 AI 优化** - 专门为 AI 应用设计的智能节点
- **📱 响应式** - 适配不同屏幕尺寸和设备
- **⚡ 高性能** - 支持大量节点的流畅渲染
- **🛠 可定制** - 灵活的配置选项和扩展能力

## 📦 组件分类

### 核心组件

- **FlowCanvas** - 主画布组件，所有交互的基础
- **CustomNode** - 通用节点组件，支持多种类型
- **AINode** - AI专用节点，显示模型信息和状态
- **CustomEdge** - 自定义连接线，支持标签和状态

### 示例集合

- **BasicFlow** - 基础工作流示例
- **AINotesFlow** - AI笔记处理流程
- **WorkflowEditor** - 交互式流程编辑器

### 工具函数

- **useFlowState** - 状态管理Hook
- **autoLayout** - 自动布局算法
- **validateFlow** - 流程验证工具
- **导入导出** - 数据持久化功能

## 🚀 快速开始

```tsx
import { FlowCanvas, BasicFlow } from '@noteum/ui'

function App() {
  return (
    <div style={{ width: '100%', height: '500px' }}>
      <BasicFlow />
    </div>
  )
}
```
